# Diginvited Mobile App

Selamat datang di repositori resmi **Diginvited Mobile App**. Aplikasi ini merupakan pendamping (companion app) dari platform undangan digital [diginvited.cloud](https://diginvited.cloud).

> **Catatan:** Repositori ini hanya berisi dokumentasi (README) dan rilis aplikasi (APK). Source code aplikasi bersifat privat dan tidak didistribusikan di sini.

## Deskripsi

Diginvited Mobile dirancang khusus untuk mempermudah penyelenggara acara dalam mengelola daftar tamu, mempercepat proses _check-in_, dan memantau kehadiran secara _real-time_.

## Fitur Utama

- **Manajemen Tamu**: Mengelola data tamu, memperbarui status kehadiran, dan mencari nama tamu dengan cepat.
- **Pemindaian QR Code**: Validasi tiket atau undangan tamu secara instan menggunakan pemindai QR bawaan.
- **Overview Acara**: Pantau statistik kedatangan tamu melalui _dashboard_ interaktif secara _real-time_.

## Cara Instalasi

Anda dapat mengunduh file instalasi (APK) langsung dari repositori ini:

1. Buka tab **Releases** di repositori GitHub ini.
2. Unduh file berekstensi `.apk` dari versi terbaru.
3. Buka file tersebut di perangkat Android Anda (pastikan Anda mengizinkan instalasi dari _Unknown Sources_ jika diminta).
